# 🕺 Text-to-Gesture: Генерация Жестов из Текста

## 🎯 Цель проекта

Создание системы, которая **на вход получает текст** (фразу, предложение, абзац) и **выдаёт скелетную анимацию жестов**, которые бы естественным образом сопровождали речь при произнесении этого текста.

> Пример:  
> `Текст` → 🤖 `Смысловые, выразительные, плавные жесты`

Жесты должны быть **семантически согласованы**, **плавны**, **естественны** и **разнообразны**.

---

## 🧠 Подходы и Модели

Мы рассмотрели несколько подходов:

| Метод | Выразительность | Обучение | Реализация | Ресурсы |
|-------|------------------|-----------|------------|---------|
| LSTM / Seq2Seq | 🟡 Средняя | ⚡ Среднее время | ✅ Простая | 💻 Обычный GPU |
| Transformer / Attention | 🟢 Высокая | 🐢 Медленнее | ⚠️ Средняя | 🔋 Выше |
| Conditional GAN | 🟢 Реалистичность | 🔄 Нестабильно | ⚠️ Сложно | ⚡ Высоко |
| Diffusion-модели | 🟢 Потенциал | 🐌 Очень медленно | 🔥 Сложно | 🧠 Огромные |
| Мультимодальные (текст + речь) | 🟢 Лучшая согласованность | 🐌 | ⚠️ Высокая | 🔋 Выше нормы |

> ⚡ **Выбранные методы:**
> 1. `Transformer` — основа.
> 2. `Multimodal (Text + Audio)` — улучшение согласованности с речью.
> 3. `Conditional GAN` — вариант для повышения реалистичности.
